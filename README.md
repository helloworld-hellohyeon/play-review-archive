# 이매지너리 🧩

트위터/X 데이터 아카이브에서 내 트윗을 필터링하고, 스레드와 이미지를 폴더 구조의 ZIP으로 저장해주는 브라우저 기반 도구입니다.

서버로 데이터를 전송하지 않으며 모든 처리는 브라우저 내에서 이루어집니다.

---

## 구현된 기능

### 파일 업로드

- `tweets.js` / `tweets.json` 드래그 앤 드롭 및 클릭 업로드
- 트윗 수가 많아 파일이 여러 개로 분할된 경우(`tweets-part1.js` 등) 동시 선택 후 병합 처리

### 필터링

- **날짜 접두사**: `yyMMdd`/`yyyyMMdd` 형식으로 시작하는 트윗
- **사진 + 스레드**: 이미지가 첨부된 독립 트윗 중 스레드가 2개 이상인 것
- **키워드/해시태그**: 본문에 특정 문자열을 포함하는 트윗
- 리트윗(`RT @`) 자동 제거
- 중복 트윗 ID 제거 (분할 파일 병합 시 발생하는 중복 처리)

### 아카이브 생성

- 필터링된 각 트윗을 **폴더 1개**로 구성 (폴더명 = 본문 첫 줄)
- 폴더 내 `tweets.txt`: 루트 트윗 + 스레드 전문
- 폴더 내 이미지 파일: 루트 및 스레드 전체 이미지 다운로드 포함
- 동명 폴더 자동 넘버링으로 충돌 방지
- 처리 진행률 표시 (필터링 / ZIP 생성 단계별)
- 완료 후 통계 표시 (전체 트윗 수 / 루트 수 / 스레드 수)
